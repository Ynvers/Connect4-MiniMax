# "Puissance 4" like game

## Prerequisites

Install `Python 3 (>= 3.6.4)`
Install a venv if needed

## Run the game

### Create a virtualenv

```bash
    cd puissance4Like
    python -m venv venv
    source venv/bin/activate
```
### Install dependancies

```bash
    pip install -r requirements.txt
``` 

### Run it

```bash
    python3 main.py
```
